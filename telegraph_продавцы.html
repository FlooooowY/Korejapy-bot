🚀 Начало работы

Первый запуск:

  1. Найдите бота в Telegram (спросите у администратора)
  2. Нажмите /start
  3. Нажмите /menu - откроется главное меню
✅ Вам НЕ нужна регистрация - вы автоматически продавец!

━━━━━━━━━━━━━━━━━━

💰 Как начислить баллы клиенту?

Когда клиент что-то купил:

Шаг 1: Нажмите "💰 Добавить оплату"

Шаг 2: Введите данные клиента (любой вариант):

  • По юзернейму: @ivan или ivan
  • По телефону: +79991234567
  • По ID: 123456789

Шаг 3: Введите сумму: 1500

Шаг 4: Готово! ✅

📱 Пример:
Клиент купил на 2000₽

1. "💰 Добавить оплату"
2. Вводите @ivan
3. Вводите 2000
4. Готово! Клиенту +20 баллов

━━━━━━━━━━━━━━━━━━

💸 Как списать баллы клиенту?

Когда клиент хочет скидку:

  1. Нажмите "💸 Списать баллы"
  2. Введите данные клиента
  3. Бот покажет баланс
  4. Введите количество баллов
  5. Готово!

💡 Курс обмена:

5 баллов = 1 рубль скидки

Примеры:

  • 25 баллов = 5₽ скидки
  • 50 баллов = 10₽ скидки
  • 100 баллов = 20₽ скидки
⚠️ Важно:
• Нельзя списать больше чем есть
• Клиент получит уведомление

━━━━━━━━━━━━━━━━━━

🔍 Как узнать данные клиента?

Вариант 1: Спросите

"Какой у вас username в Telegram?"

Вариант 2: Попросите показать профиль

"Откройте бот → /menu → 'Мой профиль' → покажите ID"

Вариант 3: Если клиент новый

"Откройте бот → /start → зарегистрируйтесь"

❓ Частые вопросы

❓ Клиент не получил баллы

Что делать:

  1. Проверьте правильно ли ввели данные
  2. Спросите: "Вы зарегистрированы?"
  3. Попросите нажать /start
  4. Позовите администратора

❓ Не могу найти клиента

Попробуйте:

  • Ввести username БЕЗ @ (просто ivan)
  • Попросить номер телефона
  • Попросить ID из профиля

❓ Баллов мало для скидки

"У вас 30 баллов = 6₽ скидки
Нужно ещё покупок на 200₽"

❓ Ошибка "Клиент не найден"

Причины:

  • Клиент НЕ зарегистрирован
  • Неправильно введены данные
  • Клиент удалил бота

Решение: Попросите открыть бота и нажать /start
